重置版本预告

「极简主义」系列将全面升级为「简」系列

  • 原生版本将由1.12.2变为其他更高版本，同时空岛核心插件也会更换
  • 新增"空岛通行证"(不同于空岛等级) 以及 "空岛Robot" 、"公会"
  • 随通行证等级升高,可解锁更高规格的刷石机/团队规模/空岛边界/Robot等
  • 消息/计分板/菜单排版及Lore等将迎来大变化
  • 将会有更高效更系统化的插件联动，以及更多原创插件



什么是极简主义

我们致力于打造一个只有一个核心玩法的服务端
不同于其他服务端的玩法多种多样
但我相信少而精更能打动玩家
当然,由于个人能力及其他原因,未能保持该端持续更新
在这里向各位表示歉意

* 当然,每个人的审美都不一样
* 有些"圈内大佬"也重新定义了极简主义,让我们期待他的服务端


服务端特点 Features

  • 含有完全自主原创插件/JavaScript/插件系统
  • 插件信息排版简练美观一致,大部分空岛端未汉化的内容均已汉化
  • 独创功能,仅仅使用DM和DeluxeJoin就能实现进入游戏自动飞行偏好设置
  • 物品商店等菜单均为DM编写,不为凑时间凑数而使用其他不成熟插件
  • 帮助(/help)等快捷命令可视化,鼠标点击即补全命令/执行命令等操作
  • 空岛边界,让你的服务器看起来更有沉浸性
  • 新增"活动"功能&菜单,与自编权限系统&其他菜单/系统相配合 NEW
  • 圣诞节活动已准备完毕,将根据服务器时间自动开放 NEW
  • 新增"成就"功能&菜单,更好的留住玩家 NEW

  • 总结: 创新,完善,精简

注意


1.在开服前请先输入
  1. /deop AzureHanChen
  2. /lp user AzureHanChen clear
  3. /eco set AzureHanChen 0
  4. /points set AzureHanChen 0
复制代码
防止您在关闭正版验证后有部分"聪明"人士趁机夺权

2.服务器默认正版验证开启 , 如需关闭,请手动将
server.properties中的 online-mode=true 改为 online-mode=false

3.使用本服务端造成一切经济/精神损失概不负责

4.给予玩家金币/点券/权限指令
